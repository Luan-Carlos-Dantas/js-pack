  ## Variavéis
  `let`, `const`, `var`
  Os tipos de declarações variáveis possíveis são:
    - let
    - const
    - var
      ```javascript
      var example
      ```
  As variáveis possuem seus escopos definidos, que definem o contexto delas, que definem onde determinada viariável pode ou não ser acessada.
